
# EXX network coding challenge




## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`NEXT_PUBLIC_PROJECT_ID`


## How to run project

Install project dependencies

```bash
  yarn install
```

run project

```bash
  yarn dev
```
